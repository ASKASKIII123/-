# 论文格式自动修改软件 - 快速启动指南

## 🚀 快速开始（推荐）

### 最简单的方法
**双击运行 `启动软件.bat` 文件**

这个启动器会自动：
- ✅ 检查Python版本
- ✅ 检查并安装依赖包
- ✅ 验证程序文件
- ✅ 启动软件

### 备用方法
**双击运行 `run_enhanced.bat` 文件**

这是一个增强版的启动脚本，提供详细的检查和错误提示。

## 📋 使用流程

1. **选择文件** - 点击"选择文件"按钮，选择要格式化的Word或PDF文档
2. **选择模板** - 在"预设模板"标签页中选择合适的模板（可选）
3. **自定义设置** - 在其他标签页中调整字体、段落、页面布局等（可选）
4. **开始格式化** - 点击"开始格式化"按钮，选择输出位置并保存

## 🎯 预设模板说明

| 模板名称 | 适用场景 | 正文字体 | 标题字体 | 行间距 | 页边距 |
|---------|---------|---------|---------|-------|-------|
| academic | 学术论文 | 宋体12号 | 黑体16号 | 1.5倍 | 标准 |
| journal | 期刊论文 | Times New Roman 11号 | Arial 14号 | 2.0倍 | 窄边距 |
| thesis | 学位论文 | 宋体14号 | 黑体18号 | 1.5倍 | 宽边距 |
| conference | 会议论文 | 宋体10号 | 黑体12号 | 1.25倍 | 紧凑 |

## ⚠️ 重要提示

1. **备份文件** - 格式化前请备份原始文件
2. **文件格式** - 仅支持 .docx 和 .pdf 格式
3. **Word文档** - .doc 格式需要先转换为 .docx
4. **PDF文档** - PDF格式化功能相对有限

## 🔧 常见问题

### Q: 提示"无法识别pip命令"？
A: 使用 `python -m pip` 代替 `pip` 命令

### Q: 依赖包安装失败？
A: 尝试使用国内镜像源：
```bash
python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 程序无法启动？
A: 检查Python版本（需要3.7+）和依赖包是否正确安装

### Q: 格式化后文档内容丢失？
A: 确保使用正确的文件格式，建议先在简单文档上测试
