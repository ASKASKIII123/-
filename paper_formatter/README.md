# 论文格式自动修改软件

一个功能强大的桌面应用程序，用于自动修改学术论文的格式。支持Word文档（.docx）和PDF文档，提供多种预设模板和自定义格式选项。

## 功能特性

### 支持的文件格式
- Word文档 (.docx)
- PDF文档 (.pdf)

### 主要功能
1. **预设模板**
   - 学术论文模板
   - 期刊论文模板
   - 学位论文模板
   - 会议论文模板
   - IMRAD格式模板（新增）
   - CSSCI格式模板（新增）

2. **字体设置**
   - 正文字体和字号设置
   - 标题字体和字号设置
   - 支持中英文字体

3. **段落格式**
   - 行间距调整
   - 首行缩进设置
   - 段前段后间距

4. **页面布局**
   - 页边距设置（上、下、左、右）
   - 页面尺寸调整

5. **引用格式**
   - APA格式
   - MLA格式
   - CSSCI格式（新增）

6. **对比模式**（新增）
   - 绿色高亮新增内容
   - 红色删除线标记删除内容
   - 一目了然地查看格式和措辞调整

7. **分章节处理**（新增）
   - 自动识别文档章节（摘要、引言、方法等）
   - 按章节逐个优化
   - 支持中英文章节名称

## 安装步骤

### 环境要求
- Python 3.7 或更高版本
- Windows 操作系统

### 快速启动（推荐）

**双击运行 `启动软件.bat` 文件**

启动器会自动：
- ✅ 检查Python版本
- ✅ 使用清华镜像源加速安装依赖包
- ✅ 验证程序文件
- ✅ 启动软件

### 手动安装（可选）

如果需要手动安装，可以按照以下步骤：

1. 克隆或下载项目到本地

2. 进入项目目录：
```bash
cd paper_formatter
```

3. 安装依赖包（使用清华镜像源加速）：
```bash
python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn
```

## 使用方法

### 启动程序

在命令行中运行：
```bash
python main.py
```

### 操作步骤

1. **选择文件**
   - 点击"选择文件"按钮
   - 选择要格式化的Word或PDF文档

2. **选择模板**（可选）
   - 在"预设模板"标签页中选择合适的模板
   - 模板会自动应用一套标准的格式设置

3. **自定义格式设置**（可选）
   - **字体设置**：设置正文和标题的字体、字号
   - **段落格式**：调整行间距、首行缩进、段间距
   - **页面布局**：设置页边距
   - **引用格式**：选择引用格式（APA或MLA）

4. **开始格式化**
   - 点击"开始格式化"按钮
   - 选择输出文件的位置和名称
   - 等待格式化完成

## 预设模板说明

### 学术论文模板
- 正文字体：宋体，12号
- 标题字体：黑体，16号，加粗
- 行间距：1.5倍
- 首行缩进：2字符
- 页边距：上2.54cm，下2.54cm，左3.17cm，右3.17cm

### 期刊论文模板
- 正文字体：Times New Roman，11号
- 标题字体：Arial，14号，加粗
- 行间距：2.0倍
- 首行缩进：1.27cm
- 页边距：上下左右均为2.54cm

### 学位论文模板
- 正文字体：宋体，14号
- 标题字体：黑体，18号，加粗
- 行间距：1.5倍
- 首行缩进：2字符
- 页边距：上下左右均为3.0cm

### 会议论文模板
- 正文字体：宋体，10号
- 标题字体：黑体，12号，加粗
- 行间距：1.25倍
- 首行缩进：2字符
- 页边距：上2.0cm，下2.0cm，左2.5cm，右2.5cm

### IMRAD格式模板
- 正文字体：Times New Roman，12号
- 标题字体：Arial，14号，加粗
- 行间距：1.5倍
- 首行缩进：1.27cm
- 页边距：上下左右均为2.54cm
- 说明：IMRAD（Introduction, Methods, Results, and Discussion）是国际通用的学术论文结构

### CSSCI格式模板
- 正文字体：宋体，12号
- 标题字体：黑体，16号，加粗
- 行间距：1.5倍
- 首行缩进：2字符
- 页边距：上2.54cm，下2.54cm，左3.17cm，右3.17cm
- 引用格式：自动应用CSSCI引用格式（宋体10.5号）
- 说明：CSSCI（中文社会科学引文索引）是中国社会科学领域的重要数据库

## 新功能说明

### 对比模式

**功能介绍：**
对比模式可以让你一目了然地看到格式化前后的变化。

**使用方法：**
1. 选择要格式化的文件
2. 在"对比模式"标签页中勾选"启用对比模式"
3. 点击"选择原始文件"按钮，选择格式化前的原始文件
4. 选择格式设置并开始格式化

**显示效果：**
- 🟢 **绿色文字**：表示新增的内容
- 🔴 **红色删除线**：表示删除的内容

**适用场景：**
- 修改论文格式后，需要对比修改内容
- 审查他人修改的文档
- 确认格式化是否正确

### 分章节处理

**功能介绍：**
分章节处理功能允许你按章节逐个优化，而不是一次性处理全文。

**支持的章节：**
- 中文章节：摘要、引言、方法、结果、讨论、结论、参考文献
- 英文章节：Abstract、Introduction、Methods、Results、Discussion、Conclusion、References
- 数字章节：第一章、第二章、第三章等

**使用方法：**
1. 选择要格式化的文件
2. 在"章节处理"标签页中勾选"启用分章节处理"
3. 点击"刷新章节列表"按钮
4. 在章节列表中选择要格式化的章节（可多选）
5. 选择格式设置并开始格式化

**优势：**
- ✅ 精确控制每个章节的格式
- ✅ 避免一次性处理全文可能带来的问题
- ✅ 可以针对不同章节应用不同格式

**注意事项：**
- 章节识别基于关键词匹配，可能不完全准确
- 建议在格式化后检查章节内容
- 未选择的章节将保持原格式

## 注意事项

1. **文件备份**
   - 建议在格式化前备份原始文件
   - 程序会创建新的格式化文件，不会覆盖原文件

2. **Word文档**
   - 仅支持.docx格式，不支持.doc格式
   - 格式化后请检查文档内容是否完整

3. **PDF文档**
   - PDF格式化功能相对有限
   - 主要支持页边距调整和页码添加

4. **引用格式**
   - 引用格式化功能会自动识别以[数字]开头的引用
   - 请确保引用格式正确
